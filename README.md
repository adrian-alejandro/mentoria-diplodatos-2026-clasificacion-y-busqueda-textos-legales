# mentoria-diplodatos-2026-clasificacion-y-busqueda-textos-legales
Proyecto de mentoría de la Diplomatura en Ciencia de Datos de la UNC para desarrollar un pipeline de NLP y Machine Learning aplicado a textos legales argentinos (SAIJ). El sistema permitirá agrupar documentos por temática, predecir su fuero judicial y responder consultas mediante búsqueda semántica.
